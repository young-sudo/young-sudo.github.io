Snakemake

Snakemake is a Python-based workflow engine for building reproducible, scalable, and portable data pipelines. It uses Make-like rules, automatically manages dependencies, and supports parallel execution on local machines, clusters, and cloud backends. Its YAML/JSON configuration, conda/Apptainer integration, and powerful DAG engine make it ideal for bioinformatics pipelines.

Quick Practical Guide

  • Create a Snakefile with rules describing inputs, outputs, and commands.
  • Use wildcards to generalize rules across many samples.
  • Define a config file (config.yaml) to separate parameters from logic.
  • Run locally with snakemake -j or submit to HPC using --cluster profiles.
  • Use conda: or container: in rules for reproducible environments.

For more details, visit the official Snakemake documentation.
