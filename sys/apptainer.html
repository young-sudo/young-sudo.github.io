Singularity/Apptainer

Apptainer is a container runtime designed for HPC environments where Docker is not suitable. It runs containers as non-root users, ensuring security and compatibility on shared clusters. Apptainer can pull and convert Docker images, making it the standard container solution for scientific computing workflows.

Quick Practical Guide

  • Pull existing images: apptainer pull image.sif docker://ubuntu:22.04.
  • Run containers without root on HPC using apptainer exec or run.
  • Build .sif images from Dockerfiles for reproducibility.
  • Bind paths (e.g., --bind /data) to access HPC storage.
  • Use in workflow engines: Snakemake (container:) and Nextflow (singularity.enabled = true).

For more details, visit the official Apptainer documentation.
